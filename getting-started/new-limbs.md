# Naming Limbs

When you click Ne&#x77;**,** you'll get a popup asking you to name the limb. This is so you can differentiate between limbs within a character (such as left and right) as well as between different characters.  The layer names can always be altered later if you change your mind.

Limber will make a new set of layers and give each one a name based on a combination of the name you entered in the dialog box, and the preset you had selected in the [name preset dropdown](../the-limber-panel.md). If you selected the preset _Leg_ and specified _Right_, you’ll get _Right-Ankle_, _Right-Hip_ and _Right-Leg_.

To edit or add to the name presets, head over to the bottom of the [Settings Panel](settings-panel.md#naming-editor).&#x20;

### Renaming limbs

### &#x20;<img src="../.gitbook/assets/Button_0007_Rename.png" alt="" data-size="original">&#x20;

The **Rename** button will change the **suffixes** (_Ankle_, _Hip, Leg etc_) of all the layers in the selected limb. To do this, it uses the names from the currently selected preset in the dropdown. So in our example, we could select the _Right-Ankle_ layer, and with the _Arm_ preset selected, click Rename. Limber then renames the layers to _Right-Wrist_, _Right-Shoulder_ and _Right-Arm_.  This is really handy if you duplicate a leg, intending to make it into an arm, for example.

If you Alt-click the rename button, Limber will change the **prefix**, so you can change _Right_ to _Left_, for example.

{% hint style="info" %}
When manually renaming limbs, it's best practice to leave the dash between the prefix and the suffix.
{% endhint %}
